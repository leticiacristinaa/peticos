# Peticos - PetShop

Projeto fictício de um site para pet shop desenvolvido com HTML, CSS e JavaScript.

## 📌 Sobre o projeto

O objetivo deste projeto foi praticar:

- Estruturação com HTML
- Estilização com CSS
- Interatividade com JavaScript
- Organização de layout responsivo

## 🚀 Funcionalidades

- Página inicial
- Serviços do pet shop
- Área de contato
- Navegação interativa
- Layout responsivo

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
